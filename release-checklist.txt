# Clane Release Checklist #

Before issuing a release or pre-release, the issuer verifies each of the
following items.

* libclane_la_LDFLAGS -version-info exists and is correct.
* AC_INIT project version exists and is correct.
* Project compiles on all supported platforms.
* Tests pass on all supported platforms.
* Example applications compile and work OK.
* Doxygen documentation generates and looks OK.
* Distribution tarball contains everything needed to build project.
* Release notes are up-to-date.
* README.md is up-to-date.
* Source control repository contains all canonical files.
* Source control repository contains tag for this release.
